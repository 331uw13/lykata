# lykata
> The name comes from finnish word "lykätä", meaning: "postpone", "to delay something"

* Dependencies: NCurses, json-c

![example0](https://github.com/331uw13/lykata/blob/main/screenshots/example0.png?raw=true)


## Instructions

```
git clone https://github.com/331uw13/lykata.git
cd lykata
make
```
> Or you can use gcc_build.sh
